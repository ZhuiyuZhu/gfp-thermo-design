# 备选序列池 (Alternative Sequence Pool)

## 主序列备选方案

若主序列在实验验证或计算预测中表现不佳，可从以下备选池中替换。

### 备选池 A: 保守型备选

| 备选ID | 基础序列 | 调整 | 新突变 | 风险 | 适用场景 |
|--------|---------|------|--------|------|---------|
| Alt-A1 | Seq-1 | E222Q → E222G | E222G | 极低 | E222Q在排除列表中 |
| Alt-A2 | Seq-1 | 添加S30K | E222Q+S30K | 低 | 需要额外亮度补偿 |
| Alt-A3 | Seq-2 | N149K → N149R | E222Q+N149R | 低 | N149K在排除列表中 |

### 备选池 B: 热稳定型备选

| 备选ID | 基础序列 | 调整 | 新突变 | 风险 | 适用场景 |
|--------|---------|------|--------|------|---------|
| Alt-B1 | Seq-3 | Q183R → Q183K | E222Q+N149K+Q183K | 中 | Q183R导致亮度骤降 |
| Alt-B2 | Seq-4 | 移除M218V | E222Q+N149K+Q183R | 中 | M218V导致折叠问题 |
| Alt-B3 | Seq-4 | M218V → M218I | E222Q+N149K+Q183R+M218I | 中高 | M218V体积变化过大 |
| Alt-B4 | Seq-6 | L195A → L195S | E222Q+N149K+Q183R+L195S+S231V | 高 | L195A导致loop过柔 |

### 备选池 C: 亮度型备选

| 备选ID | 基础序列 | 调整 | 新突变 | 风险 | 适用场景 |
|--------|---------|------|--------|------|---------|
| Alt-C1 | Seq-5 | 添加N149K | E14K+E47K+E173K+E222Q+N149K | 中 | 需要增强热稳定性 |
| Alt-C2 | Seq-5 | E14K → E14R | E14R+E47K+E173K+E222Q | 中 | E14K在排除列表中 |
| Alt-C3 | Seq-2 | 添加S30R(确认) | E222Q+N149K+S30R | 低 | S30R增强折叠 |

### 备选池 D: 探索型备选

| 备选ID | 基础序列 | 调整 | 新突变 | 风险 | 适用场景 |
|--------|---------|------|--------|------|---------|
| Alt-D1 | Seq-6 | S231V → S231I | E222Q+N149K+Q183R+L195A+S231I | 高 | S231V体积变化过大 |
| Alt-D2 | Seq-6 | 添加A206K | E222Q+N149K+Q183R+L195A+S231V+A206K | 极高 | 追求极致稳定 |
| Alt-D3 | WT | 仅E222Q+F99A | E222Q+F99A | 中 | 最保守探索 |

## 快速替换决策树

```
主序列问题
    │
    ├─→ 亮度 < 30% WT
    │       └─→ 替换为保守备选 (Alt-A1, Alt-A3)
    │
    ├─→ 在排除列表中
    │       └─→ 微调突变位点 (Alt-A1, Alt-B1, Alt-C2)
    │
    ├─→ Rosetta ΔΔG > 0 (去稳定)
    │       └─→ 替换为同性质更小变化 (Alt-B3, Alt-D1)
    │
    ├─→ MD显示结构破坏
    │       └─→ 移除问题突变 (Alt-B2)
    │
    └─→ 综合表现平庸
            └─→ 尝试探索型备选 (Alt-D2)
```

## 备选序列生成脚本

```python
# 快速生成备选序列
from scripts.02_sequence_assembler_v2_1 import GFPDesigner

designer = GFPDesigner()

# 自定义突变组合
alt_mutations = {
    'Alt-B1': {221: 'Q', 148: 'K', 182: 'K'},  # Q183K instead of R
    'Alt-B2': {221: 'Q', 148: 'K', 182: 'R'},   # Remove M218V
    'Alt-A1': {221: 'G'},  # E222G instead of Q
}

for name, muts in alt_mutations.items():
    seq = designer.apply_mutations(SFGFP_WT, muts)
    print(f"{name}: {seq}")
```

## 备选序列验证

所有备选序列必须经过：
1. 长度检查 (220-250 aa)
2. 字符检查 (仅20种标准AA)
3. 发色团保护 (Y66, G67未变)
4. 排除列表比对
5. ESM-2/Rosetta快速评估
