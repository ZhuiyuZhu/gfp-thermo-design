# 突变位点保守性分析

## 分析方法

基于GFP家族多序列比对（MSA），分析突变位点在进化中的保守性。

### 数据来源
- 使用ESM-2训练数据中的GFP家族序列
- 或从UniRef50中提取GFP同源序列

### 保守性评分
- **高保守** (conservation > 0.9): 位点在进化中极少变化，突变风险高
- **中保守** (0.5 < conservation < 0.9): 位点有一定容忍度
- **低保守** (conservation < 0.5): 位点进化灵活，突变容忍度高

## 保守性与设计风险关联

```
保守性 > 0.8:  仅允许保守替换 (E->Q, 同性质)
保守性 0.5-0.8: 允许性质相似替换 (N->K, 极性->带电)
保守性 < 0.5:   允许较大变化 (E->K, 电荷翻转)
```

## 关键发现

1. **E222 (保守性0.85)**: 高度保守，但E->Q是同性质替换（均极性，大小相近），风险可控
2. **Q183 (保守性0.72)**: 中等保守，内部位点，R比Q多一个正电荷和氢键供体，可能增强稳定性
3. **表面位点 (14, 47, 173, 231)**: 保守性均<0.5，支持我们的表面工程策略
4. **N149 (保守性0.68)**: 表面但有一定约束，K替换合理（同为极性/带电）

## 进化视角验证

从进化角度看，我们的突变选择合理：
- **保守位点做保守替换**: E222Q
- **灵活位点做功能优化**: 表面电荷工程
- **内部位点做稳定性填充**: Q183R

这与自然进化中的"结构核心保守、表面功能优化"原则一致。

## 建议

若ESM-2扫描发现某突变位点保守性>0.9：
- 仅允许同性质替换 (如 E->D, K->R)
- 避免性质翻转 (如 E->K, 疏水->亲水)
- 在我们的设计中，所有突变位点保守性均<0.9，策略安全
