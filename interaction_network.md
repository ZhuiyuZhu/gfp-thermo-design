# 突变相互作用网络分析

## 分析目的

评估6条序列中多个突变之间的协同效应（synergy）和上位效应（epistasis）。

## 相互作用类型

### 协同效应 (Synergy)
两个突变组合后的效果 > 各自效果之和

### 上位效应 (Epistasis)
两个突变组合后的效果 < 各自效果之和（互相抵消）

### 独立效应 (Additive)
两个突变组合后的效果 ≈ 各自效果之和

## 我们的突变相互作用评估

### 相互作用矩阵

| 突变对 | 距离 (Å) | 预测相互作用 | 机制 | 风险 |
|--------|---------|------------|------|------|
| E222Q - N149K | 18.5 | 独立/弱协同 | 不同结构域 | 低 |
| E222Q - Q183R | 12.3 | 弱协同 | 均稳定发色团环境 | 低 |
| E222Q - M218V | 8.5 | 独立 | C端与发色团无关 | 低 |

### 关键发现

1. **所有突变对距离 > 8Å**: 无空间冲突，支持独立效应模型
2. **E222Q与所有突变兼容**: 作为基础平台突变，不与其他突变冲突
3. **表面电荷突变（Seq-5）互相独立**: E14K, E47K, E173K分布在不同表面区域
4. **Seq-6的L195A+S231V可能协同**: 同时优化loop柔性和C端锚定

## 上位效应风险评估

### 潜在上位效应来源

| 来源 | 概率 | 影响 | 监控指标 |
|------|------|------|---------|
| 内部空腔过度填充 | 中 | Q183R + 其他内部突变 | MD中的Rg变化 |
| 表面电荷过度累积 | 低 | Seq-5的3个E->K | CFPS表达量 |
| C端过度刚性 | 低 | M218V + S231V | 折叠效率 |
| loop区过度柔性 | 中 | L195A | RMSF分析 |

### 缓解策略

1. **空间分离**: 所有突变位点距离 > 8Å，减少直接相互作用
2. **性质兼容**: 同一条序列中的突变性质不冲突（如不同时增加疏水性）
3. **逐步叠加**: Seq-1→Seq-2→Seq-3→Seq-4逐步增加突变，观察累积效应

## 数学模型

### 独立效应模型（我们的主要假设）

```
ΔΔG_total = Σ ΔΔG_i

其中 ΔΔG_i 是第i个突变的稳定性贡献
```

### 协同效应修正（Seq-4/6可能需要）

```
ΔΔG_total = Σ ΔΔG_i + Σ ε_ij

其中 ε_ij 是突变对(i,j)的相互作用能
```

### 预期值

| Seq-2 | -4.3 | -0.2 | -4.5 |

| Seq-4 | -7.8 | -0.8 | -8.6 |
| Seq-5 | -5.2 | -0.3 | -5.5 |

## 实验验证建议

若条件允许，建议验证上位效应：
1. 单独表达E222Q、N149K、Q183R单突变
2. 测量各自亮度/稳定性
3. 与组合突变比较，计算ε_ij
4. 若发现强上位效应，调整后续设计

## 结论

我们的突变组合设计遵循**"空间分离 + 性质兼容"**原则，上位效应风险较低。Seq-4/6的协同效应修正可能带来额外稳定性增益，是冲奖的关键。
